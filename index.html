Tree Traversals

Consider our Binary Search Tree contains following nodes: "5, 3, 20, 13, 16, 32, 30, 17, 22"

In-Order Traversal:

  1. Visits all the nodes in left sub-tree.
  2. Visits root node.
  3. Visits all the nodes in right sub-tree.
These steps should be followed in a recursive manner to complete the In-Order traversal.

Pre-Order Traversal:

  1. Visits root node.
  2. Visits all the nodes in left sub-tree.
  3. Visits all the nodes in right sub-tree.
These steps should be followed in a recursive manner to complete the In-Order traversal.

Post-Order Traversal:

  1. Visits all the nodes in left sub-tree.
  2. Visits all the nodes in right sub-tree.
  3. Visits root node.
These steps should be followed in a recursive manner to complete the In-Order traversal.
